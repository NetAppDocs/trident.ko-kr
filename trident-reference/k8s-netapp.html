KubernetesмҷҖ нҶөн•©лҗң NetApp м ңн’Ҳ

02/09/2022 кё°м—¬мһҗ

NetApp мҠӨнҶ лҰ¬м§Җ м ңн’Ҳ нҸ¬нҠёнҸҙлҰ¬мҳӨлҠ” Kubernetes нҒҙлҹ¬мҠӨн„°мқҳ лӢӨм–‘н•ң мёЎл©ҙкіј нҶөн•©лҗҳм–ҙ кі кёү лҚ°мқҙн„° кҙҖлҰ¬ кё°лҠҘмқ„ м ңкіөн•ҳл©°, Kubernetes кө¬м¶•мқҳ кё°лҠҘ, кё°лҠҘ, м„ұлҠҘ л°Ҹ к°Җмҡ©м„ұмқ„ н–ҘмғҒн•©лӢҲлӢӨ.

м•„мҠӨнҠёлқј

"м•„мҠӨнҠёлқј" нҚјлё”лҰӯ нҒҙлқјмҡ°л“ңмҷҖ мҳЁн”„л ҲлҜёмҠӨ лӘЁл‘җм—җм„ң Kubernetesм—җм„ң мӢӨн–үлҗҳлҠ” лҚ°мқҙн„° лҰ¬м№ҳ м»Ён…Ңмқҙл„Ҳ мӣҢнҒ¬лЎңл“ңлҘј мҶҗмүҪкІҢ кҙҖлҰ¬, ліҙнҳё л°Ҹ мқҙлҸҷн•  мҲҳ мһҲмҠөлӢҲлӢӨ. AstraлҠ” нҚјлё”лҰӯ нҒҙлқјмҡ°л“ң л°Ҹ мҳЁн”„л ҲлҜёмҠӨм—җ мһҲлҠ” NetAppмқҳ кІҖмҰқлҗҳкі  кҙ‘лІ”мң„н•ң мҠӨнҶ лҰ¬м§Җ нҸ¬нҠёнҸҙлҰ¬мҳӨмқҳ TridentлҘј мӮ¬мҡ©н•ҳм—¬ мҳҒкө¬ м»Ён…Ңмқҙл„Ҳ мҠӨнҶ лҰ¬м§ҖлҘј н”„лЎңл№„м ҖлӢқн•ҳкі  м ңкіөн•©лӢҲлӢӨ. лҳҗн•ң, мҠӨлғ…мғ·, л°ұм—… л°Ҹ ліөмӣҗ, нҷңлҸҷ лЎңк·ё, лҚ°мқҙн„° ліҙнҳё, мһ¬н•ҙ/лҚ°мқҙн„° ліөкө¬, лҚ°мқҙн„° к°җмӮ¬, Kubernetes мӣҢнҒ¬лЎңл“ңмқҳ л§Ҳмқҙк·ёл Ҳмқҙм…ҳ мӮ¬мҡ© мӮ¬лЎҖлҘј мң„н•ң м•ЎнӢ°лёҢ нҒҙлЎ  ліөм ңмҷҖ к°ҷмқҖ н’Қл¶Җн•ң кі кёү м• н”ҢлҰ¬мјҖмқҙм…ҳ мқёмӢқ лҚ°мқҙн„° кҙҖлҰ¬ кё°лҠҘмқ„ м ңкіөн•©лӢҲлӢӨ.

ONTAP

ONTAPлҠ” лӘЁл“  м• н”ҢлҰ¬мјҖмқҙм…ҳм—җ кі кёү лҚ°мқҙн„° кҙҖлҰ¬ кё°лҠҘмқ„ м ңкіөн•ҳлҠ” NetAppмқҳ лӢӨмӨ‘ н”„лЎңнҶ мҪң нҶөн•© мҠӨнҶ лҰ¬м§Җ мҡҙмҳҒ мІҙм ңмһ…лӢҲлӢӨ. ONTAP мӢңмҠӨн…ңмқҖ All-Flash, н•ҳмқҙлёҢлҰ¬л“ң лҳҗлҠ” All-HDD кө¬м„ұмқ„ м ңкіөн•ҳл©° м—”м§ҖлӢҲм–ҙл§Ғ н•ҳл“ңмӣЁм–ҙ(FAS л°Ҹ AFF), нҷ”мқҙнҠёл°•мҠӨ(ONTAP Select), нҒҙлқјмҡ°л“ң м „мҡ©(Cloud Volumes ONTAP) л“ұ лӢӨм–‘н•ң кө¬м¶• лӘЁлҚёмқ„ м ңкіөн•©лӢҲлӢӨ.

TridentлҠ” мң„м—җ м–ёкёүлҗң лӘЁл“  ONTAP л°°нҸ¬ лӘЁлҚёмқ„ м§Җмӣҗн•©лӢҲлӢӨ.

Cloud Volumes ONTAP

"Cloud Volumes ONTAP" лҠ” нҒҙлқјмҡ°л“ңм—җм„ң ONTAP лҚ°мқҙн„° кҙҖлҰ¬ мҶҢн”„нҠёмӣЁм–ҙлҘј мӢӨн–үн•ҳлҠ” мҶҢн”„нҠёмӣЁм–ҙ м „мҡ© мҠӨнҶ лҰ¬м§Җ м–ҙн”Ңлқјмқҙм–ёмҠӨмһ…лӢҲлӢӨ. Cloud Volumes ONTAPлҘј мҡҙмҳҒ мӣҢнҒ¬лЎңл“ң, мһ¬н•ҙ ліөкө¬, DevOps, нҢҢмқј кіөмң  л°Ҹ лҚ°мқҙн„°лІ мқҙмҠӨ кҙҖлҰ¬м—җ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мҠӨнҶ лҰ¬м§Җ нҡЁмңЁм„ұ, кі к°Җмҡ©м„ұ, лҚ°мқҙн„° ліөм ң, лҚ°мқҙн„° кі„мёөнҷ”, м• н”ҢлҰ¬мјҖмқҙм…ҳ м •н•©м„ұмқ„ ліҙмһҘн•ЁмңјлЎңмҚЁ м—”н„°н”„лқјмқҙмҰҲ мҠӨнҶ лҰ¬м§ҖлҘј нҒҙлқјмҡ°л“ңлЎң нҷ•мһҘн•©лӢҲлӢӨ.

NetApp ONTAPмҡ© Amazon FSx

"NetApp ONTAPмҡ© Amazon FSx" мқҖ мҷ„лІҪн•ҳкІҢ кҙҖлҰ¬лҗҳлҠ” AWS м„ңл№„мҠӨлЎң, кі к°қмқҙ NetAppмқҳ ONTAP мҠӨнҶ лҰ¬м§Җ мҡҙмҳҒ мІҙм ңм—җм„ң м ңкіөн•ҳлҠ” нҢҢмқј мӢңмҠӨн…ңмқ„ мӢңмһ‘н•ҳкі  мӢӨн–үн•  мҲҳ мһҲлҸ„лЎқ м§Җмӣҗн•©лӢҲлӢӨ. ONTAPмҡ© FSXлҘј мӮ¬мҡ©н•ҳл©ҙ кі к°қмқҙ мқөмҲҷн•ң NetApp кё°лҠҘ, м„ұлҠҘ л°Ҹ кҙҖлҰ¬ кё°лҠҘмқ„ нҷңмҡ©н•ҳлҠ” лҸҷмӢңм—җ AWSм—җ лҚ°мқҙн„°лҘј м ҖмһҘн•  л•Ң к°„нҺём„ұ, лҜјмІ©м„ұ, ліҙм•Ҳ, нҷ•мһҘм„ұмқ„ нҷңмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. ONTAPмҡ© FSXлҠ” ONTAPмқҳ лӢӨм–‘н•ң нҢҢмқј мӢңмҠӨн…ң кё°лҠҘкіј кҙҖлҰ¬ APIлҘј м§Җмӣҗн•©лӢҲлӢӨ.

Element мҶҢн”„нҠёмӣЁм–ҙ

"мҡ”мҶҢ" мҠӨнҶ лҰ¬м§Җ кҙҖлҰ¬мһҗк°Җ м„ұлҠҘмқ„ ліҙмһҘн•ҳкі  лӢЁмҲңнҷ”лҗң мҠӨнҶ лҰ¬м§Җ м„Өм№ҳ кіөк°„мқ„ нҷңмҡ©н•ҳм—¬ мӣҢнҒ¬лЎңл“ңлҘј нҶөн•©н•  мҲҳ мһҲлҸ„лЎқ м§Җмӣҗн•©лӢҲлӢӨ. ElementлҠ” мҠӨнҶ лҰ¬м§Җ кҙҖлҰ¬мқҳ лӘЁл“  мёЎл©ҙмқ„ мһҗлҸҷнҷ”н•ҳлҠ” APIмҷҖ кІ°н•©лҗҳм–ҙ мҠӨнҶ лҰ¬м§Җ кҙҖлҰ¬мһҗк°Җ лҚ” м ҒмқҖ л…ёл ҘмңјлЎң лҚ” л§ҺмқҖ мһ‘м—…мқ„ мҲҳн–үн•  мҲҳ мһҲкІҢ н•©лӢҲлӢӨ.

NetApp HCI

"NetApp HCI" мқјмғҒм Ғмқё мһ‘м—…мқ„ мһҗлҸҷнҷ”н•ҳкі  мқён”„лқј кҙҖлҰ¬мһҗк°Җ ліҙлӢӨ мӨ‘мҡ”н•ң кё°лҠҘм—җ м§‘мӨ‘н•  мҲҳ мһҲлҸ„лЎқ н•ҳм—¬ лҚ°мқҙн„° м„јн„°мқҳ кҙҖлҰ¬ л°Ҹ нҷ•мһҘмқ„ лӢЁмҲңнҷ”н•©лӢҲлӢӨ.

NetApp HCIлҠ” Tridentм—җм„ң мҷ„м „нһҲ м§Җмӣҗн•©лӢҲлӢӨ. TridentлҠ” м»Ён…Ңмқҙл„Ҳнҷ”лҗң м• н”ҢлҰ¬мјҖмқҙм…ҳм—җ лҢҖн•ң мҠӨнҶ лҰ¬м§Җ мһҘм№ҳлҘј кё°ліё NetApp HCI мҠӨнҶ лҰ¬м§Җ н”Ңлһ«нҸјм—җ м§Ғм ‘ н”„лЎңл№„м ҖлӢқ л°Ҹ кҙҖлҰ¬н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

SANtricity

NetAppмқҳ E-Series л°Ҹ EF-Series мҠӨнҶ лҰ¬м§Җ н”Ңлһ«нҸјмқҖ лҘј мӮ¬мҡ©н•©лӢҲлӢӨ "SANtricity" м–ҙл– н•ң к·ңлӘЁмқҳ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җлҸ„ мҠӨнҶ лҰ¬м§Җ м„ңл№„мҠӨлҘј м ңкіөн•  мҲҳ мһҲлҠ”, лӣ°м–ҙлӮң к°Җмҡ©м„ұкіј м„ұлҠҘмқ„ к°–м¶ҳ к°•л Ҙн•ң мҠӨнҶ лҰ¬м§ҖлҘј м ңкіөн•ҳлҠ” мҡҙмҳҒ мІҙм ңмһ…лӢҲлӢӨ.

TridentлҠ” м ңн’Ҳ нҸ¬нҠёнҸҙлҰ¬мҳӨм—җм„ң SANtricity ліјлҘЁмқ„ мғқм„ұн•ҳкі  кҙҖлҰ¬н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

Azure NetApp Files

"Azure NetApp Files" NetAppм—җм„ң м ңкіөн•ҳлҠ” м—”н„°н”„лқјмқҙмҰҲкёү Azure нҢҢмқј кіөмң  м„ңл№„мҠӨмһ…лӢҲлӢӨ. Azureм—җм„ң кё°ліём ҒмңјлЎң к°ҖмһҘ к№ҢлӢӨлЎңмҡҙ нҢҢмқј кё°л°ҳ мӣҢнҒ¬лЎңл“ңлҘј мӢӨн–үн•ҳкі  NetAppм—җм„ң кё°лҢҖн•ҳлҠ” м„ұлҠҘ л°Ҹ к°•л Ҙн•ң лҚ°мқҙн„° кҙҖлҰ¬лҘј м ңкіөн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

AWS нҷҳкІҪмқ„ мң„н•ң Cloud Volumes Service

"AWS нҷҳкІҪмқ„ мң„н•ң NetApp Cloud Volumes Service" лҠ” NFS л°Ҹ SMBлҘј нҶөн•ҙ NAS ліјлҘЁмқ„ All-Flash м„ұлҠҘмңјлЎң м ңкіөн•ҳлҠ” нҒҙлқјмҡ°л“ң л„ӨмқҙнӢ°лёҢ нҢҢмқј м„ңл№„мҠӨмһ…лӢҲлӢӨ. мқҙ м„ңл№„мҠӨлҘј мӮ¬мҡ©н•ҳл©ҙ кё°мЎҙ м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ нҸ¬н•Ён•ң лӘЁл“  мӣҢнҒ¬лЎңл“ңлҘј AWS нҒҙлқјмҡ°л“ңм—җм„ң мӢӨн–үн•  мҲҳ мһҲмҠөлӢҲлӢӨ. ECS(Elastic Container Service) мқёмҠӨн„ҙмҠӨм—җ лҢҖн•ҙ мқјкҙҖлҗң кі м„ұлҠҘ, мҰүк°Ғм Ғмқё нҒҙлЎ  мғқм„ұ, лҚ°мқҙн„° ліҙнҳё л°Ҹ ліҙм•Ҳ м•Ўм„ёмҠӨлҘј м ңкіөн•ҳлҠ” мҷ„м „ кҙҖлҰ¬нҳ• м„ңл№„мҠӨлҘј м ңкіөн•©лӢҲлӢӨ.

Google Cloudмҡ© Cloud Volumes Service

"Google Cloudмҡ© NetApp Cloud Volumes Service" лҠ” NFS л°Ҹ SMBлҘј нҶөн•ҙ NAS ліјлҘЁмқ„ All-Flash м„ұлҠҘмңјлЎң м ңкіөн•ҳлҠ” нҒҙлқјмҡ°л“ң л„ӨмқҙнӢ°лёҢ нҢҢмқј м„ңл№„мҠӨмһ…лӢҲлӢӨ. мқҙ м„ңл№„мҠӨлҘј мӮ¬мҡ©н•ҳл©ҙ кё°мЎҙ м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ нҸ¬н•Ён•ң лӘЁл“  мӣҢнҒ¬лЎңл“ңлҘј GCP нҒҙлқјмҡ°л“ңм—җм„ң мӢӨн–үн•  мҲҳ мһҲмҠөлӢҲлӢӨ. GCE(Google Compute Engine) мқёмҠӨн„ҙмҠӨм—җ лҢҖн•ң мқјкҙҖлҗң кі м„ұлҠҘ, мҰүк°Ғм Ғмқё ліөм ң, лҚ°мқҙн„° ліҙнҳё л°Ҹ ліҙм•Ҳ м•Ўм„ёмҠӨлҘј м ңкіөн•ҳлҠ” мҷ„м „ кҙҖлҰ¬нҳ• м„ңл№„мҠӨлҘј м ңкіөн•©лӢҲлӢӨ.
